Какие языки вы должны учить
HTML5 HTML - это язык, который используется для создания и разметки веб-страниц, размещаемых в сети. Он был основан на единственной идее. Она заключалась во внесении в обычный текст ссылок.
JavaScript CSS – это язык, который позволяют вносить уникальные изменения и делать невероятный дизайн страниц, используя довольно простой код написания.
CSS3 JavaScript – это язык программирования для добавление различных анимационных эффектов, которые не представляется реализовать с помощью HTML и CSS.
PHP PHP – это высокоуровневый язык программирования, с помощью которого создают сайты, активно взаимодействующие с пользователем
